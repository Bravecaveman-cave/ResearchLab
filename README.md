# ResearchLab

> Build once. Research everywhere.

ResearchLab is an open, reproducible research platform for CFD, combustion, HPC and AI.

---

## Vision

ResearchLab aims to provide a unified research environment for:

- CFD
- Combustion
- GPU Computing
- AI
- HPC
